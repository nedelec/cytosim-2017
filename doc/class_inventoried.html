Inventoried Class Reference

Detailed Description

Inventoried provides a serial-number of type Number, used to identify objects in the simulation. A serial-number is strictly positive, and it is given only once in each class.

Inventoried [and any derived class] can be registered in a Inventory. The Inventory keeps track of all assigned serial-numbers, and can be used to find back an object of a given serial-number.
